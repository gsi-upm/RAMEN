Sentiments
==========
In order to be able to identify the sentiment of the agent, it has been chosen to change the agent's color. The colors that represent each sentiment are:

* Happiness: yellow.
* Anger: red.
* Fear: green
* Disgust: purple.
* Surprise: light blue.
* Sadness: dark blue.
